Correct output is not the same as auditable output.本轮审计发现：FinRobot 报告中的三项历史财务数值或可复算比例均未被一手来源反驳；其中 2 项 strictly supported，1 项 supported with display-precision caveat。报告自身仍不足以让读者独立追踪这些结论的证据路径。当前 3/3 not contradicted 只描述已满足 L2 条件的 claim 集合；不代表完整 NVDA 报告准确率，不代表被测系统的整体表现，也不代表 generation comparison 结论。

Overview / 项目概览

Problem

AI 可以生成看似专业的 equity research report，但可能出现数字或期间错误、估值指标缺少日期与口径、推断强于证据、正文缺乏可独立复核来源链。

Evaluation Response

本项目不重新生成报告，而是抽取可核验 claims、映射公开一手来源、确定性评分可复核项目，并对证据不闭合的 claims 明确降级。

Product Value

该方法帮助产品和评测团队判断输出是否可核验、结论是否越过证据边界、人工 reviewer 应集中在哪里、哪些失败应由 harness 提前拦截。

Product Goal / 服务目标

What product problem does this audit framework address?

专业金融 Agent 的输出不能只满足“读起来像研究报告”，还需要满足数字可以回到来源、估值指标可以重建口径、披露事实与分析推断可区分、证据不足时拒绝进入确定性结论、审核人员能快速定位真正需要人工复核的部分。

角色需要解决的问题
Agent Product / Eval Team判断专业报告输出是否可靠，并形成可复用 benchmark
Engineer获得明确、可实现、可校验的 evidence-gate 规则
Human Reviewer只集中复核高风险或证据不闭合 claims
Enterprise User in High-Responsibility Tasks获得可追踪、可解释、不夸大证据强度的输出

Product Requirement Statement

面向 AI 生成的专业报告，建立一套 evaluator workflow：将报告表述转化为绑定证据的 claims；对可复核事实进行确定性评分；对无法闭合口径的指标主动降级；将错误与缺口转化为产品改进要求，同时避免对整体报告可靠性作过度声称。

Non-Goals / 当前不服务的目标

产品解释

The audited claims are not contradicted after external primary-source verification; two are strictly supported and one is supported with a display-precision caveat. The report itself does not provide strong claim-level traceability.

因此，内容正确与报告可审计性被视为两个独立评测结果。

From Report Sentence to Auditable Claims / 从原报告表述到可核验 Claims

FinRobot 原文：Revenue surged from $26.9B in FY2023 to $60.9B in FY2024, representing a 125.9% year-over-year increase.
Extracted ClaimWhat Is VerifiedWhy It Is Separated
B01 — Revenue ValuesFY2023 与 FY2024 的营收绝对数值原始财务事实可以回溯至 10-K，但展示精度与截断规则必须单独处理。
B02 — Revenue Growth125.9% 的同比增长率派生计算必须基于精确输入重新计算

同一句报告表述可能同时包含原始事实与派生计算；将其拆分后，evaluator 才能分别判断“数字是否正确”和“计算是否可复现”。

Display Precision Caveat. B01 不再作为普通 NUM_ABS_001 支持项展示。FY2023 source value 为 $26.974B，报告展示为 $26.9B；相对差异约 0.274%，超过 0.1% relative tolerance。该项仅在 DISPLAY_PRECISION_TRUNC_001 下被视为 supported with display-precision caveat：$26.974B 截断到一位小数为 $26.9B，但普通四舍五入会得到 $27.0B。

Supported Number ≠ Fully Supported Narrative

Report ContentCurrent Audit Treatment
“SG&A as a percentage of revenue declining from 9% in FY2023 to 2.1% in FY2026”可由披露数字复算，进入 Formal L2，判断为 Supported within the predefined 0.1 percentage-point tolerance。
“Operational leverage is evident”属于分析推断，不随数字 claim 自动获得支持。
“Supporting substantial contribution margin improvement”需要额外证据与人工复核，不计入当前 deterministic score。
本框架只对证据实际能够支持的主张范围作出判断，不将数字正确性自动扩张为完整投资叙事正确性。Numeric claim supported does not mean the operational leverage inference was scored.

Rule-Based Downgrade Cases

Why Downgrade Matters / 为什么降级本身是正确行为。 Downgraded 不表示 claim 已被证明错误，而表示：该 claim 的指标定义、来源或依赖输入尚不足以支持确定性评分。将其排除在准确率分母之外，可以避免 evaluator 对未经闭合的金融指标作出虚假确定判断。
